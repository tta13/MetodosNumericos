# Métodos Numéricos
Implementando alguns métodos numéricos em Python para o projeto da disciplina. CIn-UFPE 2019.2.


##		Métodos implementados:

 - Euler
 - Euler Inverso
 - Euler Aprimorado
 - Runge-Kutta
 - Adams-Bashforth
 - Adams-Moulton
 - Função Inversa
 
##		Bibliotecas utilizadas:
 - Sympy
 - Matplotlib
